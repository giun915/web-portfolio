# Web Portfolio

Vue 3 기반으로 리팩토링한 개인 웹 포트폴리오 프로젝트입니다.
기존 퍼블리싱 작업을 기반으로 구조 개선 및 컴포넌트화 작업을 진행하였습니다.

## Overview

- 기존 HTML/CSS 퍼블리싱 프로젝트를 Vue 3로 전환
- 컴포넌트 단위 구조 설계
- 유지보수성과 확장성을 고려한 구조 개선
- Swiper 기반 섹션 네비게이션 구현
- 커스텀 마우스 휠 제어 로직 구현

## Tech Stack

- HTML5 / CSS3
- JavaScript (ES6+)
- Vue 3 (Composition API)
- Vite
- Swiper
